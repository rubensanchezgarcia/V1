# Airport Management Tool – Versión 2

Somos el Grupo 11, formado por Jana, Pablo y Rubén. Este repositorio contiene la entrega de la versión 2 del proyecto de Informática 1.

En esta versión hemos ampliado la gestión de aeropuertos desarrollada en la versión 1 para incluir también la gestión de vuelos de llegada al aeropuerto de Barcelona-El Prat (LEBL), junto con nuevas opciones de visualización y representación de datos.

El proyecto permite cargar aeropuertos y vuelos desde ficheros, añadir y eliminar aeropuertos, guardar información en archivos de salida, clasificar aeropuertos y vuelos según si pertenecen o no al espacio Schengen, generar gráficos con matplotlib y representar aeropuertos y rutas en archivos KML para su visualización en Google Earth.

Además, se ha desarrollado una interfaz gráfica en Tkinter desde la que se pueden ejecutar las principales funcionalidades del programa de forma visual y más cómoda para el usuario.

Archivos principales del proyecto:

- `airport.py`: gestión de aeropuertos
- `aircraft.py`: gestión de vuelos
- `interfaz versión 2.py`: interfaz gráfica
- `airport_test.py` y `airport_test2.py`: pruebas de funciones
- `Airports.txt` y `Arrivals.txt`: ficheros de entrada
- `schengen_Airports.txt`, `mapAirports.kml` y `routes.kml`: ficheros de salida

Para ejecutar el proyecto, basta con abrir el archivo de la interfaz y lanzar el programa desde Python.
